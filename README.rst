Genomic Surveillance is a collection of statistical algorithms to monitor local prevalence of viral lineages.


*************
Quick install
*************

Via Pypi
========

To install :code:`genomicsurveillance` via Pypi simply type

.. code-block:: console

    $ pip install genomicsurveillance

into your shell.
